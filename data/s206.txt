<h4 id="approach-1-iterative-accepted">Approach #1 (Iterative) [Accepted]</h4>
Assume that we have linked list 1 → 2 → 3 → Ø</code>, we would like to change it to Ø ← 1 ← 2 ← 3</code>.</p>
While you are traversing the list, change the current node's next pointer to point to its previous element. Since a node does not have reference to its previous node, you must store its previous element beforehand. You also need another pointer to store the next node before changing the reference. Do not forget to return the new head reference at the end!</p>
</span>public</span> ListNode</span> reverseList</span>(</span>ListNode</span> head</span>)</span> {</span>
    ListNode</span> prev</span> =</span> null</span>;</span>
    ListNode</span> curr</span> =</span> head</span>;</span>
    while</span> (</span>curr</span> !=</span> null</span>)</span> {</span>
        ListNode</span> nextTemp</span> =</span> curr</span>.</span>next</span>;</span>
        curr</span>.</span>next</span> =</span> prev</span>;</span>
        prev</span> =</span> curr</span>;</span>
        curr</span> =</span> nextTemp</span>;</span>
    }</span>
    return</span> prev</span>;</span>
}</span>

<h4 id="approach-2-recursive-accepted">Approach #2 (Recursive) [Accepted]</h4>
The recursive version is slightly trickier and the key is to work backwards. Assume that the rest of the list had already been reversed, now how do I reverse the front part? Let's assume the list is: n1</sub> → … → nk-1</sub> → nk</sub> → nk+1</sub> → … → nm</sub> → Ø</p>
Assume from node nk+1</sub> to nm</sub> had been reversed and you are at node nk</sub>.</p>
n1</sub> → … → nk-1</sub> → nk</sub></b> → nk+1</sub> ← … ← nm</sub></p>
We want nk+1</sub>’s next node to point to nk</sub>.</p>
So,</p>
nk</sub>.next.next = nk</sub>;</p>
Be very careful that n1</sub>'s next must point to Ø. If you forget about this, your linked list has a cycle in it. This bug could be caught if you test your code with a linked list of size 2.</p>
</span>public</span> ListNode</span> reverseList</span>(</span>ListNode</span> head</span>)</span> {</span>
    if</span> (</span>head</span> ==</span> null</span> ||</span> head</span>.</span>next</span> ==</span> null</span>)</span> return</span> head</span>;</span>
    ListNode</span> p</span> =</span> reverseList</span>(</span>head</span>.</span>next</span>);</span>
    head</span>.</span>next</span>.</span>next</span> =</span> head</span>;</span>
    head</span>.</span>next</span> =</span> null</span>;</span>
    return</span> p</span>;</span>
}</span>
